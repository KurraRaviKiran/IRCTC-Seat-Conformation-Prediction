# 🚆 Railway Ticket Confirmation Prediction System

## Overview

The Railway Ticket Confirmation Prediction System is a Machine Learning-based web application that predicts whether a railway ticket is likely to be confirmed based on passenger and journey details.

The project uses historical railway booking data, data preprocessing, and a Random Forest Classifier to provide confirmation predictions through an interactive Streamlit dashboard.

## Features

* Ticket confirmation prediction
* Interactive Streamlit dashboard
* Railway booking analytics and visualizations
* Passenger age distribution analysis
* Train type and quota analysis
* Confirmation status insights
* User-friendly interface

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Plotly

## Machine Learning Model

* Algorithm: Random Forest Classifier
* Task: Binary Classification
* Target Variable: Confirmation Status
* Model Accuracy: ~68%

## Dataset Features

* Date of Journey
* Class of Travel
* Quota
* Source Station
* Destination Station
* Booking Date
* Number of Passengers
* Age of Passengers
* Booking Channel
* Travel Distance
* Number of Stations
* Travel Time
* Train Type
* Special Considerations
* Holiday or Peak Season

## Dashboard Components

* Confirmed Tickets KPI
